# 📄# AutoScan 文件重點分析系統

> **⚠️ Known Issues (Demo Version)**
> 1. **File Import**: 目前 PDF 與 Word 檔案匯入功能暫時無法運作，請直接貼上文字內容進行分析。
> 2. **OpenAI Agent**: OpenAI Agent 模式暫時無法連結，請使用 Gemini 模型或 OpenAI 一般模式。

## 專案簡介📋 專案概述

AutoScan 是一個智能文件分析系統，利用最新的 AI 模型（Gemini 與 OpenAI）為您快速提取文件中的關鍵重點。系統支援多種輸入方式與檔案格式，並能無縫整合至 Notion 資料庫，是提升工作效率的強力助手。

---

## ✨ 主要功能

### 1. 📂 多格式文件支援
系統支援直接上傳以下格式進行分析：
- **TXT**：純文字檔案
- **Word (.docx)**：自動解析文檔內容
- **PDF**：智能提取文字資訊

### 2. 📝 標籤式輸入介面
- **上傳文件**：拖放或點擊即可載入檔案
- **文字貼上**：直接貼上逐字稿或筆記（至少 50 字元）

### 3. 🤖 AI 語言模型選擇
您可以根據需求選擇不同的 AI 提供商與模型版本：
- **Gemini**：包含 2.0 Flash, 2.0 Flash Thinking, 2.5 Pro 等最新模型
- **OpenAI**：支援 GPT-4o, GPT-4o Mini, GPT-4 Turbo 等核心模型

### 4. 🔗 Notion 智慧整合
- **一鍵轉存**：直接將分析出的重點建立為 Notion To-Do 項目
- **自動驗證**：即時檢查與 Notion 資料庫的連接狀態

### 5. 🎨 極致視覺體驗
- **星空粒子背景**：動態散射星空粒子，營造現代科技感
- **玻璃擬態 UI**：夢幻的毛玻璃卡片設計，支援各種螢幕解析度滿版顯示

## 🔐 安全與配置說明

為了保護您的隱私，AutoScan 採用了**安全配置系統**。敏感資訊如 API 金鑰不會儲存在網頁原始碼中。

### 📁 配置文件結構
```
AutoScan/
├── config.js         # ⚠️ 核心配置 (金鑰存放在此，不要分享！)
├── config.json       # 可選配置 (備份用)
├── .gitignore        # 防止敏感檔案被上傳至 Git
└── README.md         # 說明文件
```

### 🚀 快速設定步驟
1. 確保根目錄存在 `config.js`。
2. 在 `config.js` 中填入您的金鑰：
```javascript
window.AUTOSCAN_CONFIG = {
  "gemini": { "apiKey": "您的金鑰" },
  "openai": { "apiKey": "您的金鑰", "assistantId": "您的ID" },
  "notion": { "token": "您的Token", "databaseId": "您的ID" }
};
```
3. 重新整理頁面，系統會自動在背景讀取配置。

## 🛠️ 技術架構

### 前端技術棧
- **核心**：HTML5, CSS3, Vanilla JavaScript (無框架依賴)
- **視覺**：CSS 變數系統 + 玻璃擬態設計 + 動態 Background 渲染
- **第三方函式庫**：
  - **JSZip**：負責 Word (.docx) 檔案解析
  - **PDF.js**：負責 PDF 文字提取

### API 支援
- **Google Gemini API**
- **OpenAI Chat Completions API**
- **Notion API v1**

## 📖 使用指南

1. **開啟程式**：直接在瀏覽器開啟 `index.html`（支援 `file://` 直接啟動）。
2. **選擇語言**：在上方選擇分析目標語言。
3. **輸入內容**：
   - 點擊上傳區域選擇檔案。
   - 或切換頁籤貼上文字內容。
4. **模型設定**：
   - 在下方「語言模型」區塊切換提供商。
   - 選擇適合的模型版本。
5. **開始分析**：點擊「🔍 分析文字」。
6. **結果處理**：
   - 點擊「📋 複製重點」取得結果。
   - 點擊「📤 上傳至 Notion」同步至您的工作區。
